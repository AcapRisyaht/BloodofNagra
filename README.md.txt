# KnightOfSpirit 🎮🐉

**KnightOfSpirit** is an indie 2D action RPG game developed using Unity 2021.3 LTS.  
Inspired by Malay mythology and forest legends, you play as a young warrior bound to a hereditary spirit dragon named Nagra.

---

## 🔥 Features
- Pixel art visual style (64x64)
- Mysterious magical forest setting
- Boss fights with creatures based on Asian folklore
- Dragon transformation system
- Hand-crafted lore and solo development

---

## 💻 Tech
- Unity 2021.3 LTS
- Krita (Pixel Art)
- GitHub for version control
- Built on a mid-range laptop (HP 14-am036tx)

---

## 🎯 Project Status
This is a work-in-progress (WIP). A playable demo is planned.
